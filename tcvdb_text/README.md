# Tencent VectorDB Java SDK

Java SDK for [Tencent VectorDB Sparse Encoder](https://cloud.tencent.com/product/vdb).

## Getting started


### Prerequisites

    -   Java 8 or higher
    -   Apache Maven or Gradle/Grails

### Install Java SDK

You can use **Apache Maven** or **Gradle**/**Grails** to download the SDK.

   - Apache Maven

       ```xml
        <dependency>
            <groupId>com.tencent.tcvdb</groupId>
            <artifactId>tcvdb_text</artifactId>
            <version>1.0.0</version>
        </dependency>
       ```

   - Gradle/Grails

        ```gradle
        compile 'com.tencent.tcvectordb:tcvdb_text:1.0.0'
        ```
